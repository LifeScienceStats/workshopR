## Internal R & Git Workshop

Hello everyone!

This repository contains files that were used for the internal workshop dedicated to working with R & Git in the context of statistical analysis of clincial data.

The workshop was structured as follows:

* Day 1: Basic syntax, classes, objects, functions
* Day 2: Base & tidyverse for data analyses
* Day 3: Graphs, tables and reporting with RMarkdown / Quarto
* Day 4: Git & RStudio
